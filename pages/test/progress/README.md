## progress组件
进度条描述的是一种加载的状态，比如软件升级下载进度， 视频，图片下载进度等

### 主要属性：
```
属性名                 类型              默认值             描述
percent               Float                             百分比0~100
showInfo              Boolean          false            在进度条右侧显示百分比
strokeWidth           Number            6               进度条的宽度(px)
color                 Color            #09BB07          进度条颜色
active                Boolean          false            进度条从左往右的动画
```
